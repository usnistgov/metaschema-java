  1. Java Tools
  2. Metaschema Java Code Generator

This module provides a Java code generator Java API that generates Java classes for a Metaschema definition. These generated classes can be used to read and write XML, JSON, and YAML content that is valid to the associated Metaschema model using the binding parser library.

The following plugin can be added to your POM to use this library.

<dependency>
  <groupId>gov.nist.secauto.metaschema</groupId>
  <artifactId>metaschema-java-codegen</artifactId>
  <version>0.8.1</version>
</dependency>
